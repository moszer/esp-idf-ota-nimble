
PODX FIRMWARE SOURCE

This firmaware made for PODX
## BUILD

Command to build
```bash
 Idf.py build
```
Command to flash
```bash
 Idf.py flash
```
Command to monitor
```bash
 Idf.py monitor
```
Full command to test
```bash
 Idf.py fullclean build flash monitor
```
